Testing CSP in Chrome

I am a locked-down website, with the most stringent CSP policy: "default-src 'none';"

Let's test if a Chrome extension can inject remote script, style and iframe elements.

Install this project's extension, then click the top-right 'hello' button. Now check for success:

  • Script injection: next statement will be set 'YES': NO
  • Sandboxed script injection: next statement will be set 'YES': NO
  • Css injection: next statement will be coloured green: green?
  • Iframe injection: will be 'sidebar' iframe on page left
  • For transitive cases, where extension-injected javascript then injects elements itself:
    • Transitive script injection: next statement will be set 'YES': NO
    • Transitive css injection: next statement will be coloured yellow: yellow?
    • Transitive iframe injection: will be 'sidebar' iframe on page right
